rule34Py.pool

This module provides classes representing the Rule34 website’s Pool objects.

class rule34Py.pool.Pool(pool_id: int, name: str, description: str, posts: list[int] = <factory>)

A collection of Rule34 Posts.

Parameters:

  • pool_id (int) – The pool’s unique numeric ID.

  • name (str) – The pool’s title.

  • description (str) – A summary description of the pool’s contents.

  • posts (list[int]) – A list of Post ID numbers that are members of the pool.
