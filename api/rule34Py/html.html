rule34Py.html

This module contains classes for parsing HTML pages from the rule34.xxx site.

class rule34Py.html.ICamePage(html: str)

The Rule34 ‘icame’ page.

https://rule34.xxx/index.php?page=icame

This class can be instantiated as an object that automatically parses the useful information from the page’s html, or used as a static class to parse the page’s html directly.

Parameters:

html (str) – The page HTML to parse.

top_chart: list[ICame] = []

The top icame results, in descending order. ie. element 0 is the most popular tag.

static top_chart_from_html(html: str) → list[ICame]

Parse the ICame Top 100 chart from the page html.

Parameters:

html (str) – The ICame page HTML, as a string.

Returns:

A list of the top 100 ICame characters and their counts.

Return type:

list[ICame]
